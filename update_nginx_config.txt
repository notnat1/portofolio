Berikut adalah perintah untuk memperbarui konfigurasi Nginx Anda:

**Langkah 1: Perbarui file konfigurasi Nginx**

Jalankan perintah ini di VPS Anda. Perintah ini akan memperbarui file `/etc/nginx/sites-available/portofolio.natte.site` dengan `proxy_pass` yang lebih eksplisit (`127.0.0.1`):

```bash
echo 'ignaciojanis' | sudo -S bash -c "cat << 'EOF' > /etc/nginx/sites-available/portofolio.natte.site
server {
    listen 80;
    listen [::]:80;
    server_name portofolio.natte.site;

    root /home/natte/portofolio/client/dist;
    index index.html index.htm;

    location / {
        try_files \$uri \$uri/ /index.html;
    }

    location /api {
        proxy_pass http://127.0.0.1:3001; # Mengganti localhost dengan 127.0.0.1
        proxy_http_version 1.1;
        proxy_set_header Upgrade \$http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host \$host;
        proxy_cache_bypass \$http_upgrade;
    }
}
EOF"
```

**Langkah 2: Muat ulang Nginx**

Setelah file konfigurasi diperbarui, muat ulang Nginx agar perubahan diterapkan:

```bash
echo 'ignaciojanis' | sudo -S systemctl reload nginx
```

**Langkah 3: Verifikasi**

Setelah itu, coba akses kembali `https://portofolio.natte.site` di browser Anda.
